# Chip ERP

Sistema ERP backend desarrollado con Node.js, Express, TypeScript y Prisma.

## Objetivo
Crear un sistema de gestión de recursos empresariales modular y escalable.

## Módulos

### 1. Comex (Comercio Exterior)
- Gestión de **Pedimentos**.
- Control de **Partidas** (mercancías, fracciones arancelarias, valores en aduana).

## Stack Tecnológico
- Node.js & Express
- TypeScript
- Prisma ORM
